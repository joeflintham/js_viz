Adding new datasets

You can add news datasets in two steps:

  1. Export a matrix to csv: in your spreadsheet software, save the active worksheet as a Windows Comma Separated Values (.csv) file; save the .csv file in /Library/Webserver/Documents/dataviz/csv/data
  2. Run the data processor: in a terminal, navigate to the dataviz folder:
    cd /Library/Webserver/Documents/dataviz

    Then run the data processor script: 
    python process_data.py

Notes

Spreadsheet format: the processor script requires that the matrix in the csv conform to a specific layout. Open this example spreadhseet for reference.

Sector names: the processor bundles in Sector names using the csv/index/index.csv file. If you want include sector names which are missing from this index, add them to the index in the spreadsheet (see the example for reference) and save it as Windows Comma Separated Values file 'index.csv' in csv/index/
